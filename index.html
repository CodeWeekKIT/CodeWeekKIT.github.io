about CodeWeek

Where?

The event will be held in the Informatics department of the KIT. Building 50.34, Computer pool -142
Note: it's minus 142, not dash 142

What?

The week of code is an event that takes place in whole Europe you can check out the other events here. This is a little workshop that is a part of the big event. The workshop will be held in English and is absolutely free.

When?

The workshop will take place on Thursday 16/10/2014, at 1:30 PM

Agenda

 

  • 14:00 - 14:30

    A humble introduction

    We will talk a bit about why programming is important and why everyone should know how to code.

  • 14:30 - 15:00

    Learn by playing

    We will get to write and see some code.

  • 15:15 - 16:00
    16:15 - 17:00
